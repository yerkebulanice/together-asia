Описание API

Together Asia API работает поверх HTTP-протокола. Мы используем REST архитектуру, схема описывается в соответствии с OpenAPI. Коды возврата описываются соответствующими HTTP-статусами. Платформа принимает и возвращает JSON-структуры в HTTP body.

Любой запрос к API должен выполняться в кодировке UTF-8 и с указанием содержимого в формате JSON.

Content-Type: application/json; charset=utf-8

Аутентификация

POST /v2/signup HTTP/1.1
Host: api.together.asia
API-KEY: eyJ0eXAi...
Content-Type: application/json
{
"username":"admin",
"password":"admin"
}

Получение списка адресов Кризисных Центров

GET /v2/nko/list HTTP/1.1
Host: api.together.asia
API-KEY: eyJ0eXAi...
Content-Type: application/json
